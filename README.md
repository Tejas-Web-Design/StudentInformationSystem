# 👩‍💻 Student Information System

##📌  Project Overview

The Student Information System is a Java console-based application designed to manage student records efficiently. It allows users to add, view, search, update, and delete student information with data validation and file persistence.

-------------------------
## ✨ Features

- Add Student Records
- View All Students
- Search Student by ID
- Search Student by Name
- Update Student Details
- Delete Student Records
- Input Validation
- File Storage and Retrieval
- Menu-Driven Interface

------------------------------------------
## 🛠️ Technologies Used

- Java
- OOP Concepts
- ArrayList
- File Handling
- Exception Handling
- Scanner Class

  -----------------------------------------
## Project Structure

src/
├── Student.java
├── StudentManager.java
├── ValidationUtils.java
├── FileHandler.java
└── StudentInformationSystem.java

## How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java StudentInformationSystem
```
