# User Manual

## Main Menu Options

### Add Student

Allows users to enter:

- Student ID
- Name
- Age
- Grade
- Contact Information

### View All Students

Displays all stored student records.

### Search Student

Search students by:

- Student ID
- Student Name

### Update Student

Modify existing student information.

### Delete Student

Remove a student record using Student ID.

### Save Data

Stores records in a text file.

### Exit

Automatically saves data and closes application.
