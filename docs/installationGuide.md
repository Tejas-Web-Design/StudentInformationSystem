# Installation Guide

## Requirements

- Java JDK 8 or above
- Any IDE (IntelliJ IDEA, Eclipse, VS Code)

## Steps

1. Install Java JDK
2. Download project source code
3. Open terminal in project folder
4. Compile source files

```bash
javac *.java
```

5. Run application

```bash
java StudentInformationSystem
```

## Expected Output

The main menu of the Student Information System will be displayed.
